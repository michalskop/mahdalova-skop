The credibility of data journalism rests on readers not having to take our word for it — they should be able to see where a claim comes from and how we got there. This page explains how fact-checking works on this site, in its two different forms.

## Verifying our own articles

Before an article is published, it goes through an independent check — not by the author, but by a second person in the newsroom. For every checkable claim (a number, a quote, a name, a date, a comparison) we track the source it came from, favoring primary sources (documents, datasets, on-the-record statements) over secondary ones. For data journalism pieces, we additionally archive the source files and note the methodology behind any calculations — not just the resulting figure.

## The public fact-check format

For selected claims — from politicians, viral statements, numbers circulating in public debate — we use a standalone format with a color-coded verdict. We choose which claims to check based on public interest and verifiability, not on how controversial they are.

We use four verdicts:

```factcheck true
**True** – the claim matches verifiable facts and available sources.
```

```factcheck false
**False** – the claim is demonstrably false, supported by verifiable evidence.
```

```factcheck misleading
**Misleading** – the claim contains true elements, but as a whole creates a false impression – for example, it omits important context or uses a misleading comparison.
```

```factcheck unverifiable
**Unverifiable** – available sources can neither confirm nor disprove the claim.
```

For every verdict we list the sources we relied on, so readers can repeat the verification themselves.
